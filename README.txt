Project 2: Shading and Animation - Helicopter animation by Scott Bell

Keyboard and mouse controls:

KEYBOARD
--------------------------------
Object control (hold SHIFT)
    Q: Rotate active object around +x
    A: Rotate active object around -x
    W: Rotate active object around +y
    S: Rotate active object around -y
    E: Rotate active object around +z
    D: Rotate active object around -z

    UP: Translate active object +z
    DOWN: Translate active object -z
    LEFT: Translate active object +x
    RIGHT: Translate active object -x
    SLASH: Translate active object +y
    APOSTROPHE: Translate active object -y

    C: Change active object to camera
    H: Change active object to helicopter
    T: Change active object to a tree
    1: Change active object to the helicopter spotlight
    0: Change active object to the moon


Helicopter control (hold CTRL)
    UP: Tilt helicopter forward
    DOWN: Tilt helicopter backward
    LEFT: Tilt helicopter left
    RIGHT: Tilt helicopter right
    SLASH: Rotate left
    APOSTROPHE: Rotate right

    F: Increase propeller speed
    S: Decrease propeller speed

Other controls
    ESCAPE: Quits program
    0: Enables/Disables moon light source
    1: Enables/Disables helicopter spotlight light source
    SPACE: Pauses/Resumes animation

    UP: Translates camera -y
    DOWN: Translates camera +y
    LEFT: Translates camera +x
    RIGHT: Translates camera -x
    APOSTROPHE: Translates camera +z
    SLASH: Translates camera -z

    Q: Rotate active object around +x
    A: Rotate active object around -x
    W: Rotate active object around +y
    S: Rotate active object around -y
    E: Rotate active object around +z
    D: Rotate active object around -z


MOUSE
--------------------------------
CLICK and HOLD +
    Move LEFT: Rotates camera counterclockwise around world z axis
    Move RIGHT: Rotates camera clockwise around world z axis
    Scroll IN: Translates camera along its -y axis
    Scroll OUT: Translates camera along its +y axis
